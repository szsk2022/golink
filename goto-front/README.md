### 介绍：


本后台采用了Vue.jsV2，采用可视化方式管理链接

![GoTofront](https://www.sunzishaokao.com/cdn/gotolink/gitee/Gotofront.jpg)

### 快速开始：

 1.请您在服务器上安装Node.js14.00+
  
 2.进入源码目录

   `cd goto-front`
 
 3.打开终端，执行编译命令：


```
npm run build
```

编译后会生成dist文件夹，将文件夹上传到服务器，并且安装像Apache这样的Web服务器软件并且正确配置域名、网站目录，SSL证书即可访问控制台！

如果出现错误请与我们的客服联系！
